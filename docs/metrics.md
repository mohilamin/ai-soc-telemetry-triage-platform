# Metrics

- `total_events_processed`: telemetry rows processed.
- `total_alerts`: alerts emitted by detection rules.
- `total_incidents`: correlated incident count.
- `alert_to_incident_reduction_rate`: noise reduction from alert to incident.
- `duplicate_alert_rate`: duplicate alert share.
- `scenario_detection_rate`: injected scenario detection coverage.
- `false_positive_probability_average`: analyst queue average false-positive likelihood.
- `overall_soc_triage_score`: blended SOC readiness score.
